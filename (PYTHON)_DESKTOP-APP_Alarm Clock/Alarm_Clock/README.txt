Installation and Deployment:
	1. Install required libraries
		Windows (In CMD):
			python -m pip install -r requirements.txt

		Linux/RPi (In Terminal):
			python3 -m pip install -r requirements.txt
	2. Run Program
		Windows:
			Run using python3

		Linux/RPi (In Terminal):
			python3 Program.py

Reminders:
	1. Don't rename/delete Data.db
	2. Don't rename/delete Records folder
	3. Don't rename/delete assets
	4. All the files in this folder are supported by the program. Please don't delete or change the names of the files in this folder
